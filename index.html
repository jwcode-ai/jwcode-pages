How JWCode works

One engine, four layers, fully traceable.

A Java engine, two surfaces, every model. Plan-then-approve by default.

JWCode is built as a modular Java 17 application with a React frontend and Ink-based terminal UI. The agent core, tool system, and security layer are fully open-source (Apache 2.0).

50+ built-in tools, multi-agent orchestration, Plan/Act/Goal permission system — all connected through a 4-layer security and observability stack.

Quick start

From install to first AI session in 2 minutes.

No accounts, no cloud signup, no credit card. Build from source or use the one-liner.

Cross-platform: macOS, Windows, Linux
Zero config, sensible defaults, full overrides
Self-hosted, your data stays on your machine
1

Install Java 17+ & Node 18+

JWCode requires Java 17+ for the backend and Node.js 18+ for the frontend.

2

Build & Launch

mvn install -pl jwcode-core,jwcode-web -am -DskipTests then mvn exec:java -pl jwcode-web.

3

Open Web UI

React SPA loads at localhost:8080. Point it at your repo and start shipping.

4

Terminal TUI (optional)

cd ts-cli && npm run go for the Ink/React terminal interface.
